## Getting Started

> Baixando e instalando o Flutter.

https://flutter.dev/docs/get-started/install

> Extraindo para o ambiente de desenvolvimento.

```sh
$ cd ~/development
$ tar xf ~/Downloads/flutter_linux_1.17.2-stable.tar.xz
```
